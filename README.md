# lauluphine_quiz

Pequeña aplicación web de quiz (HTML/CSS/JS).

## Ejecutar localmente
- Abrir `index.html` en el navegador.

## Subir a GitHub
Voy a generar un script PowerShell que automatiza la creación del repo en GitHub (con `gh`) y el primer push.

> Nota: necesitas tener instalados `git` y `gh` y estar autenticado con `gh auth login`.
